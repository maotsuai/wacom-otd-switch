# Wacom-OTD Switch

A small utility for quickly switching between the official Wacom driver and OpenTabletDriver.

## Usage

1. Download the appropriate version of the `exe` file from GitHub Releases.
2. Move the `exe` to the location where you want to keep it long-term.
3. Double-click the program to run it.
4. Follow the setup window to complete the initial configuration.
5. Whenever you need to switch drivers afterward, find the program icon in the system tray and left-click it.
