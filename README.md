# Wacom-OTD Switch

在 Wacom 官方驱动与 OpenTabletDriver 之间快速切换的小工具。

## 使用方法

1. 从 GitHub Releases 下载对应版本的 `exe` 文件。
2. 将 `exe` 移动到你希望长期放置的位置。
3. 双击运行程序。
4. 按设置窗口提示完成首次配置。
5. 之后需要切换驱动时，在系统托盘找到程序图标并左键单击。
